# 问题

记录一些 `HTTP2` 相关问题

## 客户端不支持 h2

取决于**服务器是否同时支持 HTTP/1.1 回退**以及**协商机制的实现**。

| 场景                                               | 客户端不支持 HTTP/2                            | 结果                                        |
| :------------------------------------------------- | :--------------------------------------------- | :------------------------------------------ |
| **服务器同时支持 HTTP/1.1 和 HTTP/2**（常见配置）  | ALPN 或 Upgrade 协商 → 选择 HTTP/1.1           | ✅ **正常访问**（回退到 HTTP/1.1）           |
| **服务器仅支持 HTTP/2，不支持 HTTP/1.1**（极少见） | ALPN 协商失败（列表无 `h2`）→ 连接被拒绝或关闭 | ❌ **无法访问**（连接失败或协议错误）        |
| **明文 `h2c` 且服务器仅支持 `h2c`**                | 客户端发送普通 HTTP/1.1 请求（无 Upgrade）     | ❌ **无法访问**（服务器可能返回 400 或 505） |
