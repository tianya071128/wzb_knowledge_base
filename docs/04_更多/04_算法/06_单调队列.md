# 单调队列

## 概述

单调队列是一种特殊的队列数据结构，它能够维护队列中元素的单调性（单调递增或单调递减），从而能够在 O(1) 时间内获取最值，并在 O(n) 时间内处理整个序列。

## 特点

- 队列中的元素始终保持单调性质（递增或递减）
- 支持在常数时间内获取队列中的最大值或最小值
- 入队和出队操作维护单调性质

## 实现原理

单调队列通常基于双端队列（deque）实现，具有以下操作特性：

1. **插入操作**：新元素插入时，从队尾移除所有小于（或大于）新元素的值，然后将新元素加入队尾
2. **删除操作**：当需要删除元素时，如果该元素等于队首元素，则将其从队首移除
3. **获取最值**：队首元素即为当前窗口的最值

## 应用场景

### 1. 滑动窗口最值问题

单调队列最常见的应用场景是解决滑动窗口中的最值问题，例如：

- 求长度为 k 的所有连续子数组的最大值
- 求长度为 k 的所有连续子数组的最小值

### 2. 动态规划优化

在某些动态规划问题中，状态转移方程涉及区间最值查询，可以使用单调队列优化时间复杂度。

## 代码实现

```javascript
class MonotonicQueue {
  queue = [];
  // 使用队头指针, 代替 shift()
  // shift() 删除元素, 复杂度为 O(n)
  // 使用对头指针, 那么就为 O(1)
  leader = 0;

  /**
   * 向队列中添加元素
   * @param {number} n - 要添加的元素
   */
  push(n) {
    // 从队尾移除所有小于n的元素，以维护单调性
    while (
      this.queue.length > this.leader &&
      this.queue[this.queue.length - 1] < n
    ) {
      this.queue.pop();
    }
    this.queue.push(n);
  }

  /**
   * 获取队列中的最大值（队首元素）
   * @returns {number} 队列中的最大值
   */
  max() {
    // 队首是最大的元素
    return this.queue[this.leader];
  }

  /**
   * 从队列中移除元素
   * @param {number} n - 要移除的元素
   */
  pop(n) {
    // 只有当要移除的元素等于队首元素时才移除
    if (this.queue.length > this.leader && this.queue[this.leader] === n) {
      // 队头指针右移
      this.leader++;
    }
  }

  /**
   * 检查队列是否为空
   * @returns {boolean} 队列是否为空
   */
  isEmpty() {
    return this.queue.length === this.leader;
  }
}

/**
 * 滑动窗口最大值问题的解决方案
 * @param {number[]} nums - 输入数组
 * @param {number} k - 窗口大小
 * @returns {number[]} 每个滑动窗口的最大值
 */
function maxSlidingWindow(nums, k) {
  const result = [];
  const monotonicQueue = new MonotonicQueue();

  for (let i = 0; i < nums.length; i++) {
    // 当前索引大于等于k时，需要移除窗口外的元素
    if (i >= k) {
      monotonicQueue.pop(nums[i - k]);
    }

    // 添加当前元素
    monotonicQueue.push(nums[i]);

    // 当前窗口大小达到k时，记录最大值
    if (i >= k - 1) {
      result.push(monotonicQueue.max());
    }
  }

  return result;
}

// 示例
const nums = [1, 3, -1, -3, 5, 3, 6, 7];
const k = 3;
console.log('输入数组:', nums);
console.log('窗口大小:', k);
console.log('滑动窗口最大值:', maxSlidingWindow(nums, k));
// 输出: [3, 3, 5, 5, 6, 7]
```

## 注意事项

1. 单调队列不是直接存储原始数据，而是维护一个单调的顺序
2. 在滑动窗口问题中，需要及时清理不在当前窗口内的元素
3. 需要注意边界条件的处理，如空队列的情况
