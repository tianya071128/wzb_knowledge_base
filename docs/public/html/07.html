不添加 crossorigin 属性，正常请求资源：
<script src="https://developer.mozilla.org/static/js/main.a2cb3def.chunk.js"></script>

添加 crossorigin("" 或 anonymous) 属性，请求资源不携带凭证（cookie，客户端 SSL 证书或 HTTP 认证交换用户凭据。），控制台报跨域错误，资源请求错误：
<script src="https://developer.mozilla.org/static/js/main.a2cb3def.chunk.js" crossorigin></script>

添加 crossorigin(use-credentials) 属性，请求资源携带凭证（实测没有携带，可能是 cookie 不能跨站），控制台报跨域错误，资源请求错误：
<script src="https://developer.mozilla.org/static/js/main.a2cb3def.chunk.js" crossorigin="use-credentials"></script>
