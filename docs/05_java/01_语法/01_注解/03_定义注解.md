# 定义注解

## 基本语法

使用 `@interface` 关键字定义注解：

```java
// 元注解（修饰当前注解）
@元注解1
@元注解2
public @interface 注解名 {
    // 注解属性
    类型 属性名() default 默认值;
}
```

注解本质上是一个继承了 `java.lang.annotation.Annotation` 接口的特殊接口。

## 注解属性

注解的属性以方法的形式声明，方法名即属性名，返回值即属性类型：

```java
public @interface Config {
    String name();
    int timeout() default 30;
    String[] tags() default {};
}
```

### 支持的属性类型

只能写以下类型，不支持自定义对象、List：
- 基本类型：int / long / boolean / double
- 包装类不行，只能基础类型
- `String` 字符串
- `Class<?> 类对象`
- 枚举 `Enum`
- 注解类型
-以上任意类型数组


### 默认值

使用 `default` 关键字为属性指定默认值，有默认值的属性在使用时可以省略：

```java
@Config(name = "app") // timeout 和 tags 使用默认值
public class AppConfig {}
```
