# 使用注解

## 基本语法

注解以 @ 开头，写在类、方法、变量、参数上方，用于附加元数据：

```java
@Override
public String toString() {
    return "MyClass";
}
```

## 注解的属性

注解可以包含属性（成员变量），使用时需要为属性赋值：

```java
// 定义注解
public @interface Author {
    String name();
    int age() default 0; // 可以有默认值
}

// 使用注解
@Author(name = "张三", age = 25)
public class MyClass {}
```

### value 属性

当注解只需要传一个名为 `value` 的属性时，可以省略属性名：

```java
public @interface Path {
    String value();
}

// 以下两种写法等价
@Path(value = "/api/user")
@Path("/api/user")
```

### 数组属性

属性可以是数组类型，使用花括号传入多个值：

```java
public @interface Roles {
    String[] value();
}

@Roles({"admin", "editor"})
public void manageUsers() {}
```

当数组只需要传入一个值时，无需 `{}`：

```java
// 以下两种写法等价
@Roles({"admin"})
@Roles("admin")
```

## 注解的使用位置

注解可以应用于多种程序元素：

```java
// 类
@Deprecated
public class OldService {}

// 方法
@SuppressWarnings("unchecked")
public List<String> getList() { return null; }

// 字段
@NotNull
private String name;

// 参数
public void setName(@NotNull String name) {}

// 局部变量（Java 8+）
@NotNull String local = getValue();
```
